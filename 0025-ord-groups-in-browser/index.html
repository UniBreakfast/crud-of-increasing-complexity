Open console with Ctrl + Shift + I
    
    And manipulate record groups from there
    with function calls like these:
    

    createOneGroup(name, pos)
    readAllGroups(withRecords, indexed)
    updateOneGroup(name, newName)
    deleteOneGroup(name)
    findOneGroup(name)
    moveOneGroup(from, to)
    
    createOneRecord(groupName, recordName, pos)
    readAllRecords(groupName, indexed)
    updateOneRecord(groupName, recordName, newRecordName)
    deleteOneRecord(groupName, recordName)
    findOneRecord(groupName, recordName)
    moveOneRecord(groupName, from, to)
